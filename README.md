# astrosage-api

API to view and send horoscope as SMS from [astrosage.com](http://astrosoage.com) using Flask framework and Twilio API

## Installation
```sh
$ git clone https://github.com/sandipbgt/astrosage-api.git
$ cd astrosage-api
$ pip install -r requirements.txt
$ gunicorn app:app
```
